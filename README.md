Modifiy start.sh<br>
LC_ALL => Encoding<br>
BOT_TOKEN => The Bot token u get from the Botfather.<br>
CHAT_ID => Your Telegram ID, if u dont know the id know,<br> just add your bot, when it is up type /getchatid into your Group. <br>
SERVER_NAME => The Server Name<br>
PATH_TO_SERVER => The Path to your CSGO Server Files and screenlog.0 in the same directory<br>
SCREEN_NAME => The Name of the SCREEN THE SERVER runs in. Check with screen -ls <br>
